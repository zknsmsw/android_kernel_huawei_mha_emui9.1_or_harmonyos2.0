# android_kernel_huawei_mha_emui9.1_or_harmonyos2.0
适用于华为mate9的内核。内核编入kernelsu_0.9.2。内核源码来自华为官方。除刷入ksu外未做任何修改。
刷入前请关闭查找设备并解锁bootloader。刷入命令为：
fastboot flash kernel ksuonly_mate9_emui91_and_harmony2_kernel.img
参考文档：https://github.com/Coconutat/Huawei-GSI-And-Modify-Or-Support-KernelSU-Tutorial/wiki/7.KernelSU适配EMUI9或9.1.0系统的内核#如何为华为49版本内核集成kernelsu
